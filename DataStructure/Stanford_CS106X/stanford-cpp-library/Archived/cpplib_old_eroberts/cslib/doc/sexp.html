sexp.h

This file specifies the interface to a Lisp-like S-expression facility.
Types
sexp The abstract data type representing an S-expression object.
expression type codes This list may be expanded by the client, so the typecode must be represented as an integer and not as a closed enumeration type.

Type detail

typedef struct sexpCDT *sexp;
The abstract data type representing an S-expression object. 



This list may be expanded by the client, so the typecode
must be represented as an integer and not as a closed
enumeration type.  Expanded type codes behave like the
pointer type and may be converted to pointers.

sexp createAtom(string name);




Creates an atom with the indicated name string.  Atoms are
unique, so that a second call to createAtom with
the same string will return the same atom and will not create
a new one.  The binding and property list fields of a newly
created atom are both NIL.

@result The unique atom with the specified name

Usage:




atom = createAtom(name);
