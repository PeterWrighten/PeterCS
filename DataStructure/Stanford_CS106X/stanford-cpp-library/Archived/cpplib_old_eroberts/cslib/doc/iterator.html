bool stepIterator(Iterator iterator, void *dst);
Advances the iterator and stores the next element through the element pointer provided as a reference parameter. The stepIterator function returns true until the elements are exhausted, after which it returns false.

Usage:

hasMoreElements = stepIterator(iterator, &element);
