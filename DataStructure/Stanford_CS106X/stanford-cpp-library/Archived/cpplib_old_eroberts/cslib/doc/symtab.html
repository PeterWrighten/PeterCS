The Stanford cslib package

symtab.h

This package implements a symbol table abstraction. Clients create new key/value by calling enter and look up existing bindings by calling lookup.

void mapSymbolTable(symtabFnT fn, HashMap table, void *data);
Iterates through every entry in the symbol table and calls the function fn, passing it the following arguments:
  • The current key
  • The associated value
  • The data passed to mapSymbolTable
The data pointer allows the client to pass state information to the function fn, if necessary. If no such information is required, this argument should be NULL.

Usage:

mapSymbolTable(fn, table, data);
