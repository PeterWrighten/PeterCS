gcdebug.h

This file specifies various debugging functions for the garbage-collecting allocator.
Type
gcStateT This enumeration type holds the possible states of blocks being managed by the garbage collector:

 GCInUseBlock is in use
 GCOnFreeListBlock is on use
 GCIUnknown Block is unknown to the allocator

Functions
