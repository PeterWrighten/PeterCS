gobjects.h

This file imports a set of header files that provide access to a collection of graphical objects designed to be compatible with the ACM Graphics library for Java.

Classes

GObject This abstract class is the root of the graphics hierarchy.
GRect This class represents a rectangle.
GOval This class represents an ellipse.
GArc This class represents an elliptical arc.
GLine This class represents a line segment.
GImage This class represents a graphical image, usually read in from a file.
GLabel This class represents a text string.
GPolygon This class represents a polygon.
GCompound This class represents a compound of other graphical objects.
