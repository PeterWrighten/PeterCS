class GLabel

This class represents a text string. In addition to the ones it inherits from GObject, the GLabel class defines the following methods: 
  GLabel(string str);
  GLabel(string str, double x, double y);
  void setFont(string font);
  string getFont();
  void setLabel(string str);
  string getLabel();
  double getFontAscent();
  double getFontDescent();
