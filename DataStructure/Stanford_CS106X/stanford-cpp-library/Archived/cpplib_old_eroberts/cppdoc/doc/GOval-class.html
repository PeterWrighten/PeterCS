class GOval

This class represents an ellipse. In addition to the ones it inherits from GObject, the GOval class defines the following methods: 
  GOval(double width, double height);
  GOval(double x, double y, double width, double height);
  void setSize(double width, double height);
  void setBounds(GRectangle bounds);
  void setBounds(double x, double y, double width, double height);
  void setFilled(bool flag);
  bool isFilled();
  void setFillColor(string color);
  string getFillColor();
