class GRect

This class represents a rectangle. In addition to the ones it inherits from GObject, the GRect class defines the following methods: 
  GRect(double width, double height);
  GRect(double x, double y, double width, double height);
  void setSize(double width, double height);
  void setBounds(GRectangle bounds);
  void setBounds(double x, double y, double width, double height);
  void setFilled(bool flag);
  bool isFilled();
  void setFillColor(string color);
  string getFillColor();
