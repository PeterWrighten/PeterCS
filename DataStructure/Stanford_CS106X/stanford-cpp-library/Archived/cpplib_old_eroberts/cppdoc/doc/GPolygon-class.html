class GPolygon

This class represents a polygon. The constructor creates an empty polygon to which edges can be added. In addition to the ones it inherits from GObject, the GPolygon class defines the following methods: 
  GPolygon();
  void addVertex(double x, double y);
  void addVertex(double x, double y);
  void addEdge(double dx, double dy);
  void addPolarEdge(double r, double theta);
  void setFilled(bool flag);
  bool isFilled();
  void setFillColor(string color);
  string getFillColor();
