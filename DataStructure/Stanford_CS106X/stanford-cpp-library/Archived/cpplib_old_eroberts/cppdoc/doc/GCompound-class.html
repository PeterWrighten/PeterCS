class GCompound

This class represents a compound of other graphical objects. In addition to the ones it inherits from GObject, the GCompound class defines the following methods: 
  GCompound();
  void add(GObject gobj);
  void add(GObject gobj, GPoint pt);
  void add(GObject gobj, double x, double y);
  void remove(GObject gobj);
  void removeAll();
  int getElementCount();
