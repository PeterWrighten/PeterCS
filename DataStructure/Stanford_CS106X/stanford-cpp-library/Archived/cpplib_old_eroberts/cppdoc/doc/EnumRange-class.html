class EnumRange<BaseType>

This class is used to support iteration over enumeration types. Calling the EnumRange constructor with the first and last elements produces an iterable collection of those items, as illustrated by the following example: 
   enum Direction { NORTH, SOUTH, EAST, WEST };

   foreach (Direction dir in EnumRange(NORTH, WEST)) . . .
By convention, enumerated types exported by the libraries (such as Direction) export an enumerated range as a constant whose name is the type followed by an "s". This convention makes it possible to write this loop header as follows: 
   foreach (Direction dir in Directions) . . .
