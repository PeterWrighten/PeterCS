#include "gevents.h"

class GTableEvent : public GEvent

This event subclass represents a table event. Each mouse event records the event type (TABLE_UPDATED) along with the row, column, and cell value of the event.
Methods
getColumn() Returns the column that was updated.
getRow() Returns the row that was updated.
getValue() Returns the text in the cell that was updated.
toString() Converts the event to a human-readable representation of the event.
