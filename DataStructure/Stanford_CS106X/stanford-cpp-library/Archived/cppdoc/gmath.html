Function detail

int countDigits(int n, int base = 10);
Returns the number of digits in the given integer in the given base. Defaults to base-10, decimal. For example:
  • countDigits(3456) returns 4.
  • countDigits(0) returns 1.
  • countDigits(-778) returns 3.
  • countDigits(16, 2) returns 5.

Throws an error if base is 0 or negative.

Usage:

int digits = countDigits(16, 2);

double vectorAngle(const GPoint & pt);
double vectorAngle(double x, double y);
Returns the angle in degrees from the origin to the specified point. This function takes account of the fact that the graphics coordinate system is flipped in the y direction from the traditional Cartesian plane.

Usage:

double angle = vectorAngle(x, y);
