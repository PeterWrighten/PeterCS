The Stanford C++ Libraries

List of Modifications

The purpose of this page is to list all of the changes that have been made to the Stanford C++ libraries over time. These changes have been made primarily by current unofficial maintainer, Marty Stepp. Listing those changes here can help current developers and users of the library track down problems or issues that arise due to new or missing functionality.

  • 2014/11/13: Library now prints stack traces with line numbers on exceptions / crashes on all platforms.
  • 2014/11/13: Collection libraries updated to contain generic hashCode functions and comparison operators like < and >=. This will enable nested collections like Set<Vector<int>>.
