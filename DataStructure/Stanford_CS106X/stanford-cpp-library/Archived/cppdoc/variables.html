The Stanford C++ Libraries
Global Variables and Macros

This section lists every global variable, enum constant, and macro constant value exported by the Stanford C++ libraries, along with the name of the header file that must be included to use that value. Note that this page lists only variables that are exported into the default global namespace, not every constant member variable of every class, and not variables that are exported into any other namespaces. This also does not include overloaded operators.

You can click each column heading to sort the table by that column. For detailed documentation about a given variable, click the link for that header file.
